Note book about git

April 16, 2019

Thao tác cơ bản

Tạo một repos mới

git init

Clone 1 repos

git clone /path/to/repository

Add

Khi add file sẽ vào staging area để chuẩn bị cho lần commit tiếp theo.

Add tất cả các file.

git add .

Add những file cần thiết. Ví dụ: chỉ add những file python.

git add *.py

Sau khi add có thể remove file ra khỏi staging area.

git reset <filename>

Commit

File commit sẽ được lưu vào HEAD.

Để commit file:

git commit -m ‘message’

Để xem lịch sử commit:

git log

Push

Push những gì đã commit lên master. Có thể thay đổi master thành một branch khác.

git push origin master

Push khi chưa clone về máy.

git remote add origin <server>

Xem sự thay đổi

Chưa stage

git diff

Xem những thay đổi đã stage

git diff –cache
